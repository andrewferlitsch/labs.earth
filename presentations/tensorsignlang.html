TensorFlow - Sign Language - Digits

Abstract: The workshop is a code along for those ready to start coding their first Tensorflow neural networks. In this code along, we will build a single hidden layer neural network in TensorFlow sign language for digits 0 .. 6 from images. The code along includes using Jupyter Notebook, building layers in a neural network, activation functions, training and evaluation.

Level: Intermediate

Requirements: Basic programming knowledge of Python is required, along with general understanding of machine learning concepts.
