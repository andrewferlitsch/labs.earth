Footnotes

  1. Data type size indicator (..) represents the number of digits. The size is the maximum anticipated value when the value is a summation for the entire world.
  2. To avoid name confusion with FIPS 10-4 codes covering both countries and principal divisions, the FIPS 10-4 codes for principal administrative divisions are relabeled as FIPS 5-2 codes, which otherwise would be defined only for US principal administrative divisions.
