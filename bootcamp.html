Python Programming – Get Started (September 2017)

This job skills track teaches Python programming and programming fundaments to professionals with limited programming experience. This track covers introduction to object oriented programming, coding constructs in Python, with an introduction to coding constructs and libraries used in machine learning.

This track is ideally suited for the professional wanting to build a more solid base in programming skills coupled with experience in Python, and basic Python constructs and libraries for machine learning.

Syllabus

  • Whiteboarding - 6 hrs
    • Whiteboarding Coding Challenges in Python

    Abstract: This workshop teaches basic algorithms in whiteboarding interviews. All the code examples are in Python and the course has dual purpose teaching basic Python programming. The course covers fundamentals. Coding challenges covered are:

    • Prime Numbers
    • Fibonanci Sequences
    • Dynamic Arrays
    • Queues
    • Stacks
    • Heap
    • StackChain
    • Towers of Hanoi
    • Binary Tree
    • Binary Tree Traversals
    • BFS Traversals
    • DFS Traversals
    • Binary Tree Max/Min Depth
    • Binary Tree Max/Min Value
    • Binary Search Tree
    • Arithmetic Emulation
    • GCD
    • LCM
    • Bubble Sort
    • Insertion Sort
    • Quick Sort
    • Merge Sort
    • Hashing
    • Hashing with Linear Probing
    • Reverse String
    • Palindrome
    • Character Count
    • Duplicate Character

    Level: Fundamental

    Requirements: Some basic programming knowledge is preferred.

  • OPP Principles - 1/2 hr
    OOP Principles

    Abstract: This PDSG workshop covers the principles of OOP programming. Concepts covered are class, object, data encapsulation, inheritance, and polymorphism.

    Level: Fundamental

    Requirements: One should have some knowledge of programming.

  • Python OOP Programming - 1.2 hr
    Python OOP Programming

    Abstract: This PDSG workshop covers the basics of OOP programming in Python. Concepts covered are class, object, scope, method overloading and inheritance.

    Level: Fundamental

    Requirements: One should have some knowledge of programming.

  • Graph Theory - 1 hr
    Graph Theory

    Abstract: This PDSG workship introduces basic concepts on Tree and Graph Theory. It covers topics for level-first search (BFS), inorder, preorder and postorder depth first search (DFS), depth limited search (DLS), iterative depth search (IDS), as well as tri-coding to prevent revisiting nodes in a cyclic paths in a graph. Examples are given pictorially, pseudo code and in Python.

    Level: Fundamental

    Requirements: No prior programming knowledge is required.
